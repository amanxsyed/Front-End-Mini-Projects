## How to Use
1. Open `index.html` in a browser.
2. Press any key on your keyboard.
The app will display:
    1. The key you pressed (e.g., "A", "Enter", "Space")
    2. The key code (e.g., 65)
    3. The event code (e.g., "KeyA")
